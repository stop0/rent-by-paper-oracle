## Rent by Paper Oracle

Rental management application built with Vercel + Supabase, supporting **Owners** and **Managers**, with **Stripe** payments.

### Status
- GitHub repo created and pushed.
- Supabase schema + RLS policies added in `supabase/`.
- Next.js app scaffolded in `web/` with placeholders for auth, dashboards, and Stripe routes.

### Structure

- `supabase/`: SQL schema and RLS policies
- `web/`: Next.js (App Router) + TypeScript app for Vercel deployment
- `docs/`: deployment notes
